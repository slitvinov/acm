<h2>Install marst</h2>

<pre>
$ wget https://ftp.fau.de/gnu/marst/marst-2.7.tar.gz -O- | tar zxf -
$ cd marst-2.7
$ ./configure --prefix $HOME/.local
$ make
$ make install
</pre>

<h2>Build</h2>

<pre>
$ make
</pre>
